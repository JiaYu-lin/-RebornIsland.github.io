BEGINNING

關於資源超支

——

自始至終人類爲所欲爲的支配地球，嚴重影響地球生態資源“供给” 與“需求”的平衡，國際民間組織「全球足跡網絡」評估全球範圍 內生態承載資源消耗之極限，正伴隨著全球化的經濟趨勢急遽發展而每况愈下。

全球足跡網絡因而提出了“臨界日”的數值顯示人類正歷經“預支地球資源”生活 的險境，以今年度地球超載日統計證實，人類將於2017年8月2日超額透支地球

耗盡地球所需一年的時間“再生”才得以供應的生態資源，也意謂著將來需憑藉兩倍的地球產出資源才足以因應人類需求，而在資源消耗加快的不利條件下，超載日的提前將在所難免。

線性經濟＆循環經濟

有鑒於「地球生態超載日」的統計結果，成功促使全球社會積極發展永續的應變方案。
然則回朔環境議題的本源，人們最終體認到“經濟發展”和“環境衝擊”兩者密不 可分的因果關係。

何為導致資源衰竭主因？


自工業革命以來，地球首當其衝承受來自「線性經濟」商業發展趨勢，其開採、製造、拋棄原物料進而衍生二度環境污染。

生產端“再製”意識興起


各國組織紛紛於近幾年響應環境保護對策，以力圖改進上游產業經營，輾轉帶動「零廢棄」生產消費模式成為產業鏈循環之準則，從而減輕環境衝擊。

善用物質轉化，締造新商機


理念上有效利用資源、再製的封閉式「循環經濟」思維逐漸為生產端採納，致使廢棄物汙染 環境之通病得以改善，更其善用有害物質的轉化，締造嶄新商機。

生產消費模式改變


近幾年台灣也加入了改變生產消費模式的行列，強調經濟運作應借鏡 自然生態中的「循環」法則，生生不息的有效利用，而 不造成物質廢棄、 浪費的封閉式有機運作。

以當前時事為例，台糖公司近日將原先的畜殖場，改建為現代化畜殖場。畜殖場本於畜養時必將產生大量沼氣，而沼氣中所含“甲烷”為造成溫室效應加劇的元凶之一，其危害程度是二氧化碳的84倍



在現代科技的改良與相幫並行之下，畜殖場改為利用「沼氣發電」運作，將動物的廢棄物轉變成可再利用之能源，為畜殖場省下大筆電費，同時大幅減少對地球的危害。

SYMBIOTIC SYSTEM

生物循環–魚菜共生

——

本遊戲旨在傳達循環經濟體系中「生物循環」之理念，藉以“層級應用”發展由生物可分解原料製成產物，此外製程中廢棄的「生質原料」則經過轉化為易於利用、含有能量的生物燃料（生質能源），最後仍安全的回歸自然進而滋養生態圈。


Aquaponics 系統構成

新型複合耕作體系實現魚、蔬菜、微生物相互共養同時轉化植物可吸收肥料、淨化水源的無土有機農法。



實質效益

無土栽培，意謂無農藥用武之地。無毒農耕有效環境保護，平衡了生態系，形成可循環、低碳零排放之生產模式。

Level 1 - 魚菜共生

島上每日以隨機的方式分配有限資源，玩家若以三天打魚，兩天曬網的方式搜索糧食，生命值很快會消耗殆盡，必須取得永續資源，才能確保生存。
玩家須湊齊漁獲、水生植物、特性中空的喬木類(如：竹類植物)，並活用材料表提供之器材劈砍竹類打造魚菜共生機具，待魚菜共生規模擴建到一定程度隨即結束關卡。

遊戲開發環境

遊戲後續後台應用將延伸至online/offline虛實整合模式，提供玩家每日登入獎勵，或建立O2O營銷模式，於線上提供玩家娛樂及實質商品回饋，而線下與商業夥伴提供服務與交易上合作，玩家也可以在現實世界中得到實質的回饋。以下為當前遊戲開發環境：


雲端商業邏輯

ASP.NET：為主要的伺服器技術，資料庫存取技術使用ADO.NET實體資料模型；網路服務使用 Web API，以JSON格式做存取。促進快速發展，共用行動應用程式資源，促使應用程式因獲得雲端支援而能減少使用裝置資源。



雲端平台

Microsft Azure Web Sites服務：將資料安全的留在資料中心，集中化資料防護和桌面安全，為本遊戲主要的網站服務，透過ASP.NET將資料庫的資料存取至遊戲中。

APP行動裝置程式

目前行動裝置以Android、ios、Windows Phone為主要三大平台，其中以Android平台為最多使用者使用 ，本系統以Android為主要平台，並利用Java+Android SDK等技術開發設計Android App。



雲端資料庫

SQL Database：雲端儲存的可用性、耐久性等優勢，解決長時間傳輸以及安全疑慮。 透過資料庫存取技術ADO.NET實體資料模型與伺服器相互連結，作為主要資料庫。


消費者行為分析＆輕度休閒手遊

有鑑於當今循環經濟相關議題未能充分與手機遊戲結合達到最佳成效，於是在這款遊戲的設計上突破以往空洞的孤島求生題材做為遊戲獨創的教育寓意，以增加遊戲新鮮度，並以市場趨勢做為開發時的首要考量。



消費者遊玩時機

基於市場調研公司TNS提出「全球手遊玩家動態分析報告」，針對地域不一、年齡層18歲以上的族群施行問卷統計。指出手遊玩家最頻繁於“通勤、等候等瑣碎時間”遊玩手遊。



男女玩家喜好遊戲類型

益智、休閒類型的智慧型手遊可謂大勢所趨，透露出簡易、無負擔、便於打發時間的輕度遊戲已榮登消費者榜上必備的娛樂小品。



玩家型態改變

家長躋身為跨代遊戲玩家，其意圖包含家庭娛樂活動、教育活動等… 故本遊戲開發上針對使用者自行調理、合成、製作，寓教於樂為特色吸引小朋友、大人們一同加入遊玩。

Level 2 - 沼氣發電

玩家種植的蔬菜在設定中遭遇山豬的啃食，於是藉以原住民族狗獵山豬的捕獵方式取代捕獸夾捕捉，能免於生物遭逢截肢的命運，此時活捉的山豬則眷養起來作為沼氣來源。
鑑於綠能及環保是畜牧業永續經營的關鍵要素，玩家可於蒐集海洋漂流物或預設「材料表」中找到沼氣表等發電用具製作沼氣發電機具，即能利用沼氣發電的特點，發展生質綠能產業，最後應用於燒熱水，以及豬舍的保溫燈上。

REUSE RESOURCES

生物循環–沼氣發電

——

若將畜牧業代入生物循環框架，會形成此情境：「養豬場眷養豬隻，產銷給盤商，供給零售、消費者製造、使用食材，豬隻釋出的沼氣，此熱能來源經純化能避免額外二氧化碳釋放，“協助豬舍保溫”，導入發電機組則可“供應民生用電”。」



實質效益

豬隻排泄物經厭氧發酵槽 (digester) 排出的液體，優於傳統水肥，有效降低病菌含量、減少水質汙染、避免惡臭瀰漫空氣。



醫學領域應用

豬皮膚能夠發展出防止外傷二度感染的“豬皮敷料”。豬血則可取代胎牛血清，提煉產出液血小板濃厚血漿(P-PRP)，應用於試管嬰兒、流感疫苗、細胞培養上。甚至從人血液代用品、抗癌藥物中也能見識到豬血成分。
